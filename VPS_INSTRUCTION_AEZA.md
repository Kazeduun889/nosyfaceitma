# Инструкция по установке на VPS (Aeza)

1.  **Регистрация:**
    *   Перейдите на сайт **[aeza.net](https://aeza.net)**.
    *   Создайте аккаунт.

2.  **Заказ сервера:**
    *   Нажмите **"Виртуальные серверы"** -> **"Заказать"**.
    *   **Локация:** Выберите **Москва** (самый дешевый и быстрый пинг для СНГ) или **Хельсинки**.
    *   **Тариф:** Ищите **Promo** (если есть) или **Shared** (самый дешевый, за ~1.5 - 2 EUR/мес). 5.50 BYN (~1.6 EUR) должно хватить!
    *   **Образ (OS):** Выберите **Ubuntu 22.04**.
    *   **Оплата:** Можно оплатить картой (МИР/Visa/Mastercard) или криптовалютой. Есть почасовая оплата (для теста) или помесячная.

3.  **Получение доступа:**
    *   После оплаты (или активации теста) вам на почту придет письмо:
        *   **IP:** (например `45.14.23.11`)
        *   **User:** `root`
        *   **Password:** `какой-то-пароль`

4.  **Сообщите мне IP и пароль (или сами подключитесь):**
    *   Если сами: Скачайте **MobaXterm** (бесплатно) -> Session -> SSH -> Введите IP -> OK.
    *   В консоли введите логин `root`, потом пароль (он не виден при вводе).

5.  **Когда войдете, напишите мне "Я зашел", и я дам команду для запуска сайта!**
