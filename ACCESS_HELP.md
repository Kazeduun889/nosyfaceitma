# Инструкция для друзей: Как открыть сайт без проблем

Если сайт `scybon.pythonanywhere.com` не открывается (пишет "Не удается установить соединение" или "Превышено время ожидания"), скорее всего, ваш провайдер блокирует серверы PythonAnywhere.

Вот простые способы обойти это:

## 1. Смените DNS (Самый простой способ, без VPN)
Часто блокировка идет только по имени сайта. Смена DNS на Google или Cloudflare помогает.

**Для Windows:**
1.  Нажмите `Win + R`, введите `ncpa.cpl` и нажмите Enter.
2.  Нажмите правой кнопкой мыши на ваше подключение (Wi-Fi или Ethernet) -> **Свойства**.
3.  Найдите в списке **IP версии 4 (TCP/IPv4)** -> Выделите и нажмите **Свойства**.
4.  Выберите **"Использовать следующие адреса DNS-серверов"**:
    *   Предпочтительный: `1.1.1.1` (Cloudflare) или `8.8.8.8` (Google)
    *   Альтернативный: `1.0.0.1` или `8.8.4.4`
5.  Нажмите ОК -> ОК.
6.  Попробуйте открыть сайт.

**Для Android (без прав Root):**
1.  Скачайте приложение **1.1.1.1** (Cloudflare) из Play Market.
2.  Включите его (это не совсем VPN, он просто меняет DNS, скорость не режет).
3.  Сайт должен заработать.

## 2. Используйте VPN (Если DNS не помог)
Любой бесплатный VPN с локацией "Европа" (Нидерланды, Германия) поможет.
*   **Браузер:** Расширение "Browsec" или "Planet VPN" для Chrome/Firefox.
*   **ПК:** Программа "Planet VPN" (бесплатно).
*   **Телефон:** Любой VPN из AppStore/PlayMarket (например, ProtonVPN, Planet VPN).

## 3. Зайдите через мобильный интернет
Иногда домашний провайдер блокирует, а мобильный (МТС, Билайн, Мегафон, Tele2) — нет. Попробуйте открыть сайт с телефона через 4G.
